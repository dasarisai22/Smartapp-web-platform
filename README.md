# 🐍 SmartApp — Python + MySQL Web Application

A modern, responsive, and professional web application UI built with **HTML**, **CSS**, and **JavaScript**. This project demonstrates a full-featured frontend for a Python + MySQL backend system, complete with user authentication pages and an interactive admin dashboard.

🌐 **Live Demo:** [smartapp-web-platform.netlify.app](https://smartapp-web-platform.netlify.app)

## ⚡ Getting Started

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dasarisai22/Smartapp-web-platform.git
   ```

2. **Navigate to the project:**
   ```bash
   cd Smartapp-web-platform
   ```

3. **Open in browser:**
   - Simply open `index.html` in any web browser, OR
   - Use a local server:
     ```bash
     npx http-server -p 8080
     ```
   - Visit `http://localhost:8080`

### Deploy on Netlify

1. Push code to GitHub
2. Go to [netlify.com](https://app.netlify.com) → **Add new site** → **Import from GitHub**
3. Select the repository → Click **Deploy**
4. Your site will be live at `yoursite.netlify.app` 🎉
